This project is a car rental application. Users can filter cars based on location and make reservations for the selected vehicles. ASP.NET Core Dynamic Admin Panel Portfolio Project

This project is a dynamic admin panel and portfolio management system developed using ASP.NET Core. Through the admin panel, CRUD (Create, Read, Update, Delete) operations can be performed dynamically on database tables, statistical data can be viewed, messages can be managed as read or unread, and a to-do list can be presented in the form of notifications via the ToDoList tool. From the admin panel, we can navigate to the website page. On the homepage of my project, my personal information, skills, social media accounts, and contact details are displayed.
